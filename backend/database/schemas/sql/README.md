# Table Explanations

## Profile

- `id` - User ID
- `created_at` - Profile creation date
- `age` - User age
- `weight` - User weight
- `height` - User height
- `gender` - User gender
- `exercise` - User activity level
- `goal` - User goal weight
- `tdee` - Total Daily Energy Expenditure
- `bmr` - Basal Metabolic Rate
- `user_id` - User ID
